Flux-Vectpr Spliting

FVS (Flux-Vectpr Spliting) 矢通量分裂格式最早由 Steger-Warming 提出

  • 基本思想：将无粘通量矢量按照当地流动特征分裂为上游和下游的贡献，分别进行处理
  • 基本方法：将矢通量 Fc\bold{F_c}Fc​ （无粘通量）按 Jacobian 矩阵 Ac\bold{A_c}Ac​ 特征值符号分解为 Fc+\bold{F_c^+}Fc+​ 与 Fc−\bold{F_c^-}Fc−​ 两部分，分别采用向后差分和向前差分求解
  • 矢通量分裂格式可视为迎风格式，因为其依赖于波的传播方向，分解得到对流部分和压力部分
  • Fc+\bold{F_c^+}Fc+​ 与 Fc−\bold{F_c^-}Fc−​ 的计算方式构成不同的 FVS 格式

Fc=AcQ=Ac+Q+Ac−Q=Fc++Fc−\begin{equation} \bold{F_c} = \bold{A_c} \bold{Q} = \bold{A_c^+} \bold{Q} + \bold{A_c^-} \bold{Q} = \bold{F_c^+} + \bold{F_c^-} \end{equation}Fc​=Ac​Q=Ac+​Q+Ac−​Q=Fc+​+Fc−​​​

Ac=L−1ΛL=L−1(Λ++Λ−)L=L−1Λ+L+L−1Λ−L=Ac++Ac−\begin{equation} \bold{A_c} = \bold{L}^{-1} \bold{\Lambda} \bold{L} = \bold{L}^{-1} (\bold{\Lambda}^+ + \bold{\Lambda}^-) \bold{L} = \bold{L}^{-1} \bold{\Lambda}^+ \bold{L} + \bold{L}^{-1} \bold{\Lambda}^- \bold{L} = \bold{A_c}^+ + \bold{A_c}^- \end{equation}Ac​=L−1ΛL=L−1(Λ++Λ−)L=L−1Λ+L+L−1Λ−L=Ac​++Ac​−​​

根据 迎风原则

  • 对流通量雅克比矩阵特征值为正时，为保证稳定性，需采用向后差分格式，对应亚矢量 Fc+\bold{F_c^+}Fc+​
  • 对流通量雅克比矩阵特征值为负时，为保证稳定性，需采用向前差分格式，对应亚矢量 Fc−\bold{F_c^-}Fc−​

根据特征波传播方向

  • 特征值为正时，特征波向右传播，其依赖于左侧状态，即 Fc+(Q−)\bold{F_c^+} (\bold{Q^-})Fc+​(Q−)
  • 特征值为负时，特征波向左传播，其依赖于右侧状态，即 Fc−(Q+)\bold{F_c^-} (\bold{Q^+})Fc−​(Q+)

Van Leer’s Scheme

基本思想：通过 光滑马赫数分裂函数 进行矢通量分裂，从而构造稳定的迎风格式，以有效捕捉激波等流动现象

  • 基于马赫数选择不同分裂方式
  • 分裂函数连续可微
