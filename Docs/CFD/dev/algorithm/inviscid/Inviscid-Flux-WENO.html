WENO

主要思想

  • ENO格式 (Essentially Non-Oscillatory, 本质无振荡) 基于各种离散模板 (Stencil) 对不同的控制体计算出不同的重构多项式，选择其中最小振荡的多项式进行重构
  • WENO 对 ENO 格式的 Stencil 进行加权并结合，权重与振荡成反比，从而使得间断不会过度耗散
