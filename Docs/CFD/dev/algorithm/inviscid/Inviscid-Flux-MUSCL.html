Reconstruction

求解网格单元通量需要基于单元表面两侧物理量值计算通量差值，而通过已知离散点估计物理量空间分布的过程，称为重构 (Reconstruction)

  • 有限差分：通过网格节点上的值估计半点值
  • 有限体积（单元中心格式）：通过控制体质点值估计控制体表面值
  • 有限元：通过单元函数估计单元表面值

最简单的通量重构方式：

  • 有限差分：半点值与网格节点值一致
  • 有限体积：控制体内物理量均匀分布，内表面物理量值等于体积分平均值
